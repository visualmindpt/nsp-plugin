# NSP Default Preset

**Versão:** 1.0.0
**Autor:** NSP Plugin

## Descrição

Preset padrão do NSP Plugin com modelos base

## Instalação

1. Descarregue o ficheiro `.nsppreset`
2. Abra o NSP Control Center V2
3. Vá a "Preset Manager"
4. Clique em "Import Preset"
5. Seleccione o ficheiro descarregado

## Utilização

1. Abra Adobe Lightroom Classic
2. Seleccione as fotos que deseja editar
3. Vá ao menu do NSP Plugin
4. Escolha "AI Preset V2 - NSP Default Preset"
5. Pré-visualize e aplique!
